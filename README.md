## 🤝 Como Colaborar
Se você deseja contribuir com este projeto, siga os passos abaixo:

### Passo a passo:

1. Faça um Fork 

Clique no botão `Fork` no canto superior direito desta página. Isso criará uma cópia idêntica deste repositório na sua conta do GitHub. 

2. Clone o seu repositório

Copie a URL do seu fork e execute o comando abaixo no seu terminal (ou use o GitHub Desktop):

```
git clone https://github.com/seu-usuario/desafio-github-markdown.git
```
> Substitua o *"seu-usuario"* por o nome real do seu perfil github.

3. Fluxo de Trabalho

Para manter o projeto organizado, siga estas etapas:

**Alterações:** Realize as modificações necessárias no Markdown.

**Commits:** Salve suas alterações com mensagens claras e objetivas:

```
git add .
git commit -m "Explique brevemente o que foi feito"
git push origin sua-branch
```

4. Abra um Pull Request (PR)

Volte para a página do meu repositório original. Você verá um banner amarelo dizendo **"Compare & pull request"**. Clique nele, escreva uma breve mensagem e confirme em **"Create pull request"**.
